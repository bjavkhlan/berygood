Автотээврийн үндэсний төвийн системээс авах өгөгдлийн талаар

  • Автобусны байршлын мэдээлэл

    API-ын url руу
    GET-ээр тээврийн хэрэгслийн улсын дугаар бүхий хүсэлт явуулахад
    Тухайн агшин дахь тээврийн хэрэгслийн байршлыг илэрхийлэх lat, lng талбар бүхий өгөгдөл буцаадаг байх. 
    
	      {
	      	lat: Number,
	      	lng: Number
	      }

  • Чиглэлд явж буй тээврийн хэрэгслүүдийн мэдээлэл

    API-ын url руу
    GET-ээр хүсэлт явуулахад
    Тухайн хугацааны агшинд бүх чиглэлүүд дэх замд явж байгаа автобуснуудын мэдээллийг агуулсан өгөгдөл илгээдэг байх.
    Өгөгдөл нь дараах бүтэц бүхий JSON array байна:
    origin - хөдлөх суурин
    destination - хүрэх суурин
    waypoints - дайрч өнгөрөх суурингууд
    departures - хөдлөх цаг болон автобусны мэдээлэл 
    
	      origin - хөдлөх суурин
	      destination - хүрэх суурин
	      waypoints - дайрч өнгөрөх суурингууд 
	      departures - хөдлөх цаг болон автобусны мэдээлэл

	      [{
	      	origin: Station,
	      	destination: Station, 
	      	waypoints: [Station],
	      	departures: [{
	      	 	date:Date,
	      	 	bus:{
	      	 	 	driver: String,
	      	 	 	licenseNumber: String
	      	 	}
	      	}]
	      }]

	      /* Station */
	      Station: {
	      	name: String,
	      	lat: Number,
	      	lng: Number
	      }

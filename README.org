* Getting Started
** Prerequisities
   - NodeJS
** Installing 
   1. Clone the repository
      #+BEGIN_SRC
$ git clone https://github.com/bjavkhlan/berygood.git
      #+END_SRC
   2. Install the main components with global flag
      #+BEGIN_SRC
$ npm install -g nodemon
      #+END_SRC
   3. Install the dependencies
      #+BEGIN_SRC
$ npm install
      #+END_SRC
** Run server
      #+BEGIN_SRC
$ nodemon --ignore public/angular/
      #+END_SRC
